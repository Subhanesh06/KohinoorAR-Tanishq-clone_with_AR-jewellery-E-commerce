# 💍 Tanishq AR Jewellery E-Commerce 🚧

An interactive AR-powered jewellery e-commerce platform that allows users to explore jewellery collections and experience virtual try-on using Augmented Reality (AR).

---

# 📌 Overview

Tanishq AR Jewellery E-Commerce is a full-stack web application designed to simulate a real-world jewellery shopping experience.

It includes:
- 🛍️ User-facing e-commerce frontend
- 🧑‍💻 Admin dashboard for product management
- ⚙️ Backend API for handling data
- 🧠 AR-based jewellery try-on feature (concept/integration)

The system provides a modern, responsive interface and aims to enhance online jewellery shopping using immersive technology.
